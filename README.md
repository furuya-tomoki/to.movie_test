# to.movie
サイト名

## サイト概要
映画レビューサイト

### サイトテーマ
映画選びに困らなくなるサイト

### テーマを選んだ理由
普段あまり映画を見ないや、映画選びに悩んで結局見ないって人たちに人気の作品を伝えたい

### ターゲットユーザ
家時間多いユーザー（主に学生）

### 主な利用シーン
休日など暇な時間
