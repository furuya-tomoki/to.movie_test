## サイト概要
「to.movie」を閲覧することによって、「映画へ」もっと知れるサイトです。
また、ユーザー全員が映画レビューやお気に入り登録ができ、人気順にも閲覧できる映画情報サービスです。


### サイトテーマ
映画選びに困らなくなるサイト

### テーマを選んだ理由
普段あまり映画を見ない人や、映画選びに悩んいる人たちに人気の作品を伝えたい

### ターゲットユーザ
- 映画鑑賞が趣味のユーザ
- 家時間が多いユーザ
- 映画選択に悩んでいるユーザ
- 映画について語りたいユーザ

### 主な利用シーン
- 休日など暇な時間
- 移動時間に映画を見たいときに
- 隠れた名作をみたい時
- 仕事の合間の休憩時間などに
